# Loan Portfolio Analysis (SQL)

This project analyzes loan portfolio data using SQL, focusing on risk and performance metrics. It includes a series of SQL scripts, sample datasets, diagrams, and a sample database to demonstrate key analyses.

## Dataset
- `loan_snapshot_mock.csv` – Sample loan snapshot dataset
- `portfolio.db` – Sample database created from dummy data

## Project Structure

### SQL Scripts
- `02_portfolio_overview.sql` – Portfolio summary
- `03_movement_table.sql` – Loan movement analysis
- `04_roll_rate_calculation.sql` – Roll rate calculations
- `05_key_transition_analysis.sql` – Key transition insights
- `06_migration_matrix.sql` – Migration matrix calculations
- `07_portfolio_delinquency_rate.sql` – Delinquency rate analysis

### Diagrams / Visuals
- `delinquency_rate.png` – Visual representation of delinquency rate
- `migration_matrix.png` – Migration matrix diagram
- `portfolio_overview.png` – Portfolio overview diagram
- `roll_rate_key_transition.png` – Roll rate and key transition chart

## Key Skills Demonstrated
- SQL (joins, aggregations, CASE statements)
- Loan portfolio analysis
- Credit risk monitoring
- Data visualization

## How to Use
1. Open SQL scripts in your SQL client.
2. Run queries against `loan_snapshot_mock.csv` or `portfolio.db`.
3. Review diagrams for workflow and results.
